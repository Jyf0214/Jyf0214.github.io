The page is served as a microblog space, where I would like to publish some short and simple contents.

Instead of embedding the microblog system into Jekyll, I simply adopt the giscus and GitHub Discussions framework, so I can easily manage those short posts (and I don't need to write too many codes). Besides, I lift the comment restriction so that other users (if there are) can publish posts as well.
